Sidebar And Dashboard Example

Users will keep updates and track of each user's sales submission. Each sale will start with a "Pending" status until updated by the administrator to the following option:

  • Pending
  • Not Interested
  • No Answer
  • Fraudullant Submission
  • Re-sale Of Sales

We have also added a media query for screens that are 400px or less, which will vertically stack and center the navigation links.
